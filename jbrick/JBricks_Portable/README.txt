How to set up JBricks:

Step 1: Download the latest version of Java
	https://java.com/en/download/
Step 2: Download the LEGO Mindstorms NXT software 
	http://service.lego.com/en-us/helptopics/products/themes/mindstorms/mindstorms-nxt/nxt-software-download
Step 3: Install NXT software
Step 4: Run NXT software and update NXT Brick Firmware
	To update firmware:
		1: Plug in a NXT Brick
		2: launch click the "Tools" menu at the top of the NXT software
		3: Click "Update NXT Firmware"
		4: Select "Download"
Step 5: Launch JBricks.jar
Step 6: Set the default workspace for JBricks to use. 
	This is where files will automatically try and save to.
Step 7: Select the correct compiler for your OS. Compilers should be located within the compiler folder of JBricks.
	Windows: nbc.exe
	OS X: nbc


Note: JBricks may need to be relaunched after setting a compiler.

Enjoy using JBricks!
