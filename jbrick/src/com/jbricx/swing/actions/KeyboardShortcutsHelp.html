Keyboard Shortcuts

Note: On Windows the keyboard shortcuts will generally use "ctrl" while on OSX they will use "command"

Function Shortcut
New CTRL/COMMAND + N
Open CTRL/COMMAND + O
Save CTRL/COMMAND + S
Print CTRL/COMMAND + P
Exit CTRL/COMMAND + Q
Undo CTRL/COMMAND + Z
Redo CTRL/COMMAND + Y
Cut CTRL/COMMAND + X
Copy CTRL/COMMAND + C
Paste CTRL/COMMAND + V
Find CTRL/COMMAND + F
Go To CTRL/COMMAND + G
Select All CTRL/COMMAND + A
Preferences CTRL/COMMAND + R
Compile F5
Download F6
Find Brick CTRL/COMMAND + Shift + F
Close Current File CTRL/COMMAND + W
Forward Tab Traversal CTRL/COMMAND + right arrow key
Backward Tab Traversal CTRL/COMMAND + left arrow key
