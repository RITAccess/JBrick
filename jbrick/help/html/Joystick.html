Using the Joystick Composer

The Joystick composer will help users to operate the Brick. The use of the motors, the speed, left motor, right motor,


The Brick is able to move in all different directions like Up, Up-left, Up-right, Left, Right, Down, Down-left, Down-right.

 

The left Motor: this option allows user to set the left motor movements using the motors A, B or C

A. - It will execute just one of the motors ñ the left one.

B. - It will execute the two motors.

C. - It will execute just in case that the brick is using three motors. A, B, C.

 

The right Motor: this option allows user to set the right motor movements using the motors A, B or C

A. - It will execute only one of the motors ñ the right one

B. - It will execute the two motors.

C. - It will execute just in case that the brick is using three motors A, B, C.

 

The Controller will decide which of the two will be operated

-          None

-          xBox

-          Wii
