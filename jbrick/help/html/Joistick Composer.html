Using the Joystick Composer

When the piano composer is first opened the recording is empty. To add notes to the recording either click on the key you wish to add or press the corresponding keyboard shortcut. The keyboard shortcuts are as follows:

 


The transpose slider can be used to change the note to a higher chord.
The rest button adds a rest to the recording at the set Length.
The clear button clears all of the keys from the recording.
The copy button copies the nxc code to the clipboard that will play the stored recording.
The play button plays back the notes and rests that have been added to the recording.
The save button saves the nxc code that will play the stored recording.
The help button opens the help browser to browse the help for the piano composer.
The Note Length field specifies how long a note should play. This is modified directly by the Length factor for quickly changing how long the note will play by a divisor of 1 up to 16.
The Wait Length field specifies how long of a pause there is between notes. This is modified directly by the Length factor for quickly changing how long the note will wait by a divisor of 1 up to 16.
