JBrick Editor Features


File

New - Create a new .nxc file
Open - Open an existing .nxc file from the repository
Save - Saves the changes to an existing .nxc file
SaveAs - Saves the changes to a new .nxc file
Print - Prints the current file
Exit - Exit from the JBrick Editor

Edit

Undo - Undo the recent action or set of actions
Redo - Redo the most recent action or set of actions
Cut - Cut the selected lines of code
Copy - Copy the selected lines of code
Paste - Paste the selected lines of code to the focused location
Find - Find the result based on the keyword from

Bluetooth Troubleshooting:

If you cannot connect to a brick with your Bluetooth radio, try the following:

1)      Ensure your Bluetooth radio is enabled. This may include the wireless enabler slider on many laptops

2)      Make sure your have Bluetooth selected on the “Find Brick” GUI screen

3)      Ensure the NXT brick is installed in the devices section of your control panel

4)      Ensure you do not have multiple bricks installed in your control panel.  This sometimes causes issues within jBrick
