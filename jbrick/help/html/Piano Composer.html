Using the Piano Composer

When the piano composer is first opened the recording is empty. To add notes to the recording either click on the key you wish to add or press the corresponding keyboard shortcut. The keyboard shortcuts are as follows:

The transpose slide can be used to change the note to a higher chord. The rest button adds a rest to the recording at the set Length. Length sets the time the note will be played for or the duration of a rest. The clear button clears all of the keys from the recording. The copy button copies the nxc code to the clipboard that will play the stored recording. The play button plays back the notes and rests that have been added to the recording. The save button saves the nxc code that will play the stored recording. The help button opens the help browser to browse the help for the piano composer.
